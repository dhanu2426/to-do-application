# To-Do Application

## Overview
A simple To-Do application built with **HTML**, **CSS**, and **JavaScript**. This application allows users to create, edit, and delete tasks.

## Features
- Add new tasks.
- Edit existing tasks.
- Delete tasks.
- Mark tasks as completed.

## Technologies Used
- HTML

- JavaScript
